## 一、阶段一：骨架搭建

### 1.1 项目初始化
- 使用 Vite 初始化项目
- 安装依赖：three.js、better-sqlite3
- 创建目录结构：src/parser、src/geometry、src/interaction、src/render、server、data

### 1.2 HTML/CSS 布局
- 实现三栏布局：左侧工具栏(48px)、中间3D视口、右侧属性面板(220px)
- 底部命令输入面板(140px)
- 深色主题样式

### 1.3 Three.js 场景初始化
- 场景、相机、灯光初始化
- OrbitControls 控制器
- 渲染循环

### 1.4 坐标轴与网格
- 自定义坐标轴：X-红(右)、Y-绿(前)、Z-蓝(上)
- 坐标轴箭头和标签
- XY 平面网格

### 1.5 Tokenizer 实现
- 词法分析器，支持 IDENT/NUMBER/LPAREN/RPAREN/COMMA/EQUALS 等
- 支持希腊字母、负数、小数

### 1.6 Parser 实现
- 递归下降解析器
- 支持赋值语句、函数调用、无参命令
- 支持表达式（加减乘除）

### 1.7 ObjectStore 基础
- 对象仓库：注册/注销/查询
- 依赖图：邻接表、拓扑排序
- 序列化/反序列化

### 1.8 Executor 骨架
- 命令注册表
- 实现命令：Point、Segment、Triangle、Polygon

### 1.9 基础渲染
- GeomRenderer：点/线段/三角形/多边形渲染
- 使用 Sprite 显示标签

### 1.10 入口组装
- main.js 组装各模块
- 命令输入框事件绑定

**验收**：输入 10 行命令能看到四棱锥 ✅
