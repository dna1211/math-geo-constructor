# 几何命令生成规则

> 将文字描述转换为 3D 几何构造命令的规则文档
> 版本：v0.1（初始版本，待迭代完善）

---

## 一、坐标系约定

- **右手坐标系**：X 向右，Y 向前（屏幕里），Z 向上
- **单位**：无固定单位，默认边长为 1
- **原点**：默认居中于原点 (0, 0, 0)

---

## 二、命令语法速查

### 构造类

| 命令 | 语法 | 说明 |
|------|------|------|
| Point | `Name = Point(x, y, z)` 或 `Name = (x, y, z)` | 创建点 |
| Segment | `Segment(A, B)` | 线段（两点间） |
| Line | `Line(A, B)` | 无限直线 |
| Ray | `Ray(A, B)` | 射线（从 A 过 B 延伸） |
| Triangle | `Triangle(A, B, C)` | 三角形面 |
| Polygon | `Polygon(A, B, C, ...)` | 多边形面（≥3 点） |
| Plane | `Plane(A, B, C)` 或 `Plane(a, b, c, d)` | 平面 |

### 计算类

| 命令 | 语法 | 说明 |
|------|------|------|
| Midpoint | `M = Midpoint(A, B)` | 中点（自动跟随父对象） |
| Fold | `F = Fold(P, Line(A,B), angle)` | 绕轴旋转（角度制） |
| Reflect | `R = Reflect(P, Line(A,B))` 或 `Reflect(P, Plane(...))` | 对称/镜像 |
| Distance | `D = Distance(A, B)` | 两点距离 |

### 高级构造

| 命令 | 语法 | 说明 |
|------|------|------|
| RegularPolygon | `RegularPolygon(Segment(A,B), n, Plane(X,Y,Z), angle)` | 正 n 边形 |
| EquilateralTriangle | `EquilateralTriangle(Segment(A,B), Plane(X,Y,Z), angle)` | 正三角形 |

### 样式类

| 命令 | 语法 | 说明 |
|------|------|------|
| Color | `Color(A, "#ff0000")` | 设置颜色 |
| Dash | `Dash(A, true)` | 虚线 |
| Opacity | `Opacity(A, 0.5)` | 透明度（0-1） |
| Hide | `Hide(A)` | 隐藏 |
| Show | `Show(A)` | 显示 |
| Label | `Label(A, "文字")` | 自定义标签 |

### 操作类

| 命令 | 语法 | 说明 |
|------|------|------|
| Delete | `Delete(A)` | 删除对象 |
| Undo | `Undo` | 撤销 |
| Redo | `Redo` | 重做 |
| Clear | `Clear` | 清空全部 |
| Grid | `Grid(true/false)` | 网格开关 |
| Axis | `Axis(true/false)` | 坐标轴开关 |

## 三、命名规则

| 对象类型 | 前缀 | 示例 |
|----------|------|------|
| 点 | 大写字母 | `A`, `B`, `P1`, `V1` |
| 线段 | `seg` + 数字 | `seg1`（自动命名） |
| 直线 | `line` + 数字 | `line1`（自动命名） |
| 射线 | `ray` + 数字 | `ray1`（自动命名） |
| 三角形 | `tri` + 数字 | `tri1`（自动命名） |
| 多边形 | `poly` + 数字 | `poly1`（自动命名） |
| 平面 | `plane` + 数字 | `plane1`（自动命名） |
| 中点 | 大写字母 | `M`, `M1` |
| 正多边形顶点 | `v` + 边数 + `_` + 数字 | `v5_1`（正五边形顶点） |

**规则：**
- 用户显式赋值时使用用户指定的名称：`A = Point(0,0,0)`
- 不赋值时系统自动命名：`Segment(A, B)` → 自动命名为 `seg1`
- 正多边形的顶点由系统自动命名和创建

## 四、转换规则

### 4.1 基本几何体

#### 点

| 描述 | 命令 |
|------|------|
| "原点" / "在原点放一个点" | `O = Point(0, 0, 0)` |
| "点 (1, 2, 3)" | `A = Point(1, 2, 3)` |
| "X 轴上距离原点 3 的点" | `A = Point(3, 0, 0)` |
| "Z 轴上方 5 个单位" | `A = Point(0, 0, 5)` |

#### 线段

| 描述 | 命令 |
|------|------|
| "连接 A 和 B 的线段" | `Segment(A, B)` |
| "从原点到 (1,0,0) 的线段" | `O = Point(0,0,0)` 然后 `A = Point(1,0,0)` 然后 `Segment(O, A)` |

#### 三角形

| 描述 | 命令 |
|------|------|
| "三角形 ABC" | `Triangle(A, B, C)` |
| "边长为 2 的正三角形（XY 平面）" | `A = Point(0,0,0)` `B = Point(2,0,0)` `C = Point(1,1.732,0)` `Triangle(A,B,C)` |

#### 多边形

| 描述 | 命令 |
|------|------|
| "四边形 ABCD" | `Polygon(A, B, C, D)` |
| "正方形（边长 1，XY 平面）" | `A=(0,0,0) B=(1,0,0) C=(1,1,0) D=(0,1,0)` `Polygon(A,B,C,D)` |

#### 平面

| 描述 | 命令 |
|------|------|
| "过 A B C 三点的平面" | `Plane(A, B, C)` |
| "XY 平面" | `Plane(0, 0, 1, 0)` （方程 z=0） |
| "XZ 平面" | `Plane(0, 1, 0, 0)` （方程 y=0） |
| "YZ 平面" | `Plane(1, 0, 0, 0)` （方程 x=0） |

### 4.2 计算与变换

| 描述 | 命令 |
|------|------|
| "A 和 B 的中点" | `M = Midpoint(A, B)` |
| "A 关于线 L 对称的点" | `A' = Reflect(A, L)` |
| "A 绕轴旋转 90 度" | `A' = Fold(A, Axis, 90)` |
| "A 到 B 的距离" | `D = Distance(A, B)` |

### 4.3 样式控制

| 描述 | 命令 |
|------|------|
| "把 A 变成红色" | `Color(A, "#ff0000")` |
| "设为虚线" | `Dash(A, true)` |
| "半透明" | `Opacity(A, 0.3)` |
| "隐藏 A" | `Hide(A)` |

### 4.4 复合图形（分步构建）

对于复杂图形，需要先创建点，再组合：

**正方形：**
```
A = Point(0, 0, 0)
B = Point(1, 0, 0)
C = Point(1, 1, 0)
D = Point(0, 1, 0)
Polygon(A, B, C, D)
```

**正五边形（使用 RegularPolygon）：**
```
A = Point(0, 0, 0)
B = Point(1, 0, 0)
S = Segment(A, B)
P = Plane(0, 0, 1, 0)
RegularPolygon(S, 5, P, 0)
```

## 五、常见几何体模板

### 正四面体
```
A = Point(0, 0, 0)
B = Point(1, 0, 0)
C = Point(0.5, 0.866, 0)
D = Point(0.5, 0.289, 0.816)
Triangle(A, B, C)
Triangle(A, B, D)
Triangle(A, C, D)
Triangle(B, C, D)
```

### 正方体
```
A = Point(0, 0, 0)
B = Point(1, 0, 0)
C = Point(1, 1, 0)
D = Point(0, 1, 0)
E = Point(0, 0, 1)
F = Point(1, 0, 1)
G = Point(1, 1, 1)
H = Point(0, 1, 1)
Polygon(A, B, C, D)
Polygon(E, F, G, H)
Polygon(A, B, F, E)
Polygon(B, C, G, F)
Polygon(C, D, H, G)
Polygon(D, A, E, H)
```

### 金字塔（四棱锥）
```
A = Point(-1, -1, 0)
B = Point(1, -1, 0)
C = Point(1, 1, 0)
D = Point(-1, 1, 0)
E = Point(0, 0, 2)
Polygon(A, B, C, D)
Triangle(A, B, E)
Triangle(B, C, E)
Triangle(C, D, E)
Triangle(D, A, E)
```

### 6.7 数学题条件解析模板

**输入格式：** 用户给出题目条件描述

**解析步骤：**
1. 识别所有几何对象（点、线、面、体）
2. 识别约束条件（平行、垂直、等边、中点等）
3. 确定坐标系（原点位置、轴方向）
4. 计算各点坐标
5. 按依赖顺序生成命令（先点 → 再线/面 → 最后辅助线）

**条件→坐标映射：**

| 条件 | 处理方式 |
|------|----------|
| "正方形 ABCD" | 计算四点坐标，用 Polygon |
| "等边三角形 PAD" | 边长 L，高 L√3/2，确定第三点坐标 |
| "平面 PAD ⊥ 平面 ABCD" | ABCD 在 XY 平面，PAD 法向量在 XY 平面内 |
| "AD = 2" | 确定两点间距离为 2 |
| "E 为 PA 中点" | E = Midpoint(P, A) |
| "证明 PC // 平面 BDE" | 取辅助点 M，证明 PC // EM，EM ⊂ 平面 BDE |
