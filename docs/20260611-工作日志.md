## 四、新功能

### 主题切换（`fb149ff` + `958f506`）

在标题栏帮助按钮旁添加了 🌙/☀️ 主题切换按钮：
- **UI 层：** 20+ 个 CSS 变量覆盖（背景/边框/文字/强调色）
- **3D 场景：** 背景色 + 网格线颜色同步切换
- **持久化：** 偏好保存到 localStorage，刷新后保持
- 浅色主题配色：暖白背景 `#f5f3ee`，深色文字 `#1a1a28`

### 字符串字面量支持（`81710e4`）

tokenizer/parser/executor 全链路支持 `"..."` 和 `'...'` 字符串：
- `Color(A, "#ff0000")` 现在可以正常执行
- `Label(P, "P(1,0,√3)")` 自定义标签正确显示
