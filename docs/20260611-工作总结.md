# 2026-06-11 工作总结

## 功能开发

### 1. Line2 线条支持可变线宽和虚线
- 将 `THREE.Line` + `LineBasicMaterial` 替换为 `THREE.Line2` + `LineMaterial`（Three.js addons）
- 线宽（lineWidth）终于生效，不再受 WebGL linewidth 限制
- 虚线（dash）通过 `LineMaterial` 的 `dashed` 属性实现
- 新增 `LineWidth(obj, number)` 命令
- 属性面板新增线宽滑块和虚线复选框

### 2. 布尔值支持
- tokenizer 新增 `BOOL` token 类型（true/false）
- parser 新增 `boolean` 节点解析
- executor 新增 `boolean` 求值
- 解决了 `Dash(seg1, true)` 报错"未定义的对象: true"的问题

### 3. 属性面板改进
- 线宽滑块居中对齐（CSS 修复）
- 虚线复选框
- 可见性 checkbox 修复（之前勾选无效）

### 4. 默认样式
- Line（无限直线）和 Ray（射线）默认虚线
- Segment（线段）默认实线

## Bug 修复

### 1. 属性修改后丢失
- **根因**：`renderUpdate` 销毁重建对象，`Selector` 保存的旧材质引用失效，切换选择时旧材质被恢复
- **修复**：`Selector` 监听 `object:updated` 清理旧材质引用

### 2. 可见性 checkbox 无效
- **根因**：`renderUpdate` 未将 `style.visible` 同步到 Three.js 对象
- **修复**：在 `renderCreate` 和 `renderUpdate` 中同步 `visible` 属性

### 3. Dash 命令报错
- **根因**：tokenizer 不识别 `true`/`false`，当作标识符去 store 查找
- **修复**：新增布尔值词法分析和解析支持
