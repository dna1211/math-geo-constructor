# myjh

## 项目结构

```
├── public/              # 静态资源
├── src/
│   ├── components/      # 组件
│   ├── pages/           # 页面
│   ├── styles/          # 样式
│   ├── utils/           # 工具函数
│   └── assets/          # 资源文件
│       ├── images/
│       └── icons/
├── package.json
└── .gitignore
```

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
